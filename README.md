# 📝 To-Do Task Manager

A simple and efficient **To-Do Task Manager** application designed to help users organize their daily tasks, manage priorities, and track progress easily.

---

## 📌 Features

- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 📋 View all pending and completed tasks  
- 💾 Persistent storage (tasks are saved)
